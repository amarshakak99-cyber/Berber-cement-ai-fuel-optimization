# 🏭 Berber Cement - AI-Powered Kiln Fuel Optimization Platform

## 🌟 Overview

An advanced AI-driven platform for cement kiln optimization, developed for Berber Cement Company Ltd. The system uses machine learning to predict kiln temperature in real-time and optimize fuel mix for maximum efficiency and minimum environmental impact.

## 🤖 AI Features

- **Real-time Temperature Prediction**: Random Forest AI model predicts kiln temperature with 94.5% accuracy
- **Dynamic Fuel Optimization**: Automatically suggests optimal fuel mix (Coal, Tyre Chips, Alternative Fuels)
- **Cost & CO₂ Savings Calculator**: Real-time savings visualization
- **Predictive Analytics**: 7-day historical trend analysis

## 📊 Key Metrics

- **Target Temperature**: 1400°C
- **Update Frequency**: Every 3 seconds
- **Historical Data**: 168 hours (7 days) rolling window
- **AI Confidence**: 94.5%
